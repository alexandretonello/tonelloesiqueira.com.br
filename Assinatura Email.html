Como instalar.


    
Gmail (web)

    
    
      
  1. Clique em Copiar HTML ou selecione a assinatura renderizada e copie (Ctrl+C).
    2. 
      
  2. No Gmail: Configurações (engrenagem) → Ver todas as configurações → aba Geral.
    3. 
      
  3. Role até Assinatura → Criar novo → cole no editor (Ctrl+V).
    4. 
      
  4. Configure como padrão em "novos e-mails" e "respostas". Salvar.
    5. 
    


    
Outlook (web e desktop)

    
    
      
  1. Copie a assinatura renderizada (não o código).
    2. 
      
  2. Outlook web: Configurações → Email → Compor e responder → Nova assinatura → colar.
    3. 
      
  3. Outlook desktop: Arquivo → Opções → Email → Assinaturas.
    4. 
    


    
Apple Mail

    
    
      
  1. Preferências → Assinaturas → + e cole.
    2. 
      
  2. Desmarque "Sempre corresponder à fonte da mensagem padrão" pra preservar a tipografia.
    3. 
    


    
Quando a fonte aparecer diferente

    

      Instrument Serif e Geist são fontes web — em alguns clientes de e-mail
      mais antigos podem cair para Georgia + Arial, que é o fallback no código.
      O leitor verá uma versão um pouco menos refinada, mas as cores e a
      estrutura visual continuam intactas.
