# Tonello & Siqueira

Site institucional da **Tonello & Siqueira** — boutique brasileira de valuation,
assessoria em compra e venda de empresas e perícia judicial para pequenas e médias empresas.

🌐 **https://www.tonelloesiqueira.com.br**

## Conteúdo do repositório

| Arquivo | Função |
|---|---|
| `index.html` | A página completa (HTML, CSS e JS em um único arquivo) |
| `CNAME` | Domínio personalizado do GitHub Pages (`www.tonelloesiqueira.com.br`) |
| `.nojekyll` | Desativa o processamento Jekyll do GitHub Pages |

## Publicação (GitHub Pages)

1. Os três arquivos acima ficam na **raiz** do repositório.
2. **Settings → Pages → Branch: `main` / `(root)` → Save.**
3. Em **Custom domain**, confirmar `www.tonelloesiqueira.com.br` e marcar **Enforce HTTPS**.

## Formulário de contato

O formulário envia as mensagens por e-mail para `contato@tonelloesiqueira.com.br`
através do serviço FormSubmit (sem servidor próprio).

> **Ativação única:** no primeiro envio feito pelo site publicado, o FormSubmit
> manda um e-mail de confirmação para `contato@tonelloesiqueira.com.br`. Basta
> clicar em **Activate Form** uma vez — a partir daí, todas as mensagens chegam
> direto na caixa de entrada.
